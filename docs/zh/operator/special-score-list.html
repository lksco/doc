# 特別存譜總表

「特別存譜總表」存有除 常規樂譜 外之所有其他文件。不同於 常規存譜總表，特別存譜總表 為單一表格（無分輸入、輸出表格），故可於表格上直接進行編輯。

# 大型樂譜類別

「大型樂譜」為不能放入正規透明文件夾的樂譜，亦為唯一非放入標準文件夾的存譜分類。

# 相關連結

  • 取出樂譜 或 放入樂譜
  • 檢查樂譜：快速行動
  • 加入新譜，或，在較罕見的情況，移除譜
  • 加入狀態 或 撤回狀態

# 技術文件類別

「技術文件」為瑣碎資料或無法辨認之樂譜。於其他樂譜不同，此類文件根據「類別」存放，即同一類別的「資料」一同存於一個或多個文件夾內。

# 類別

現有適用類別如下：

類別 資料類型 例
History 任何樂團歷史資料 舊有存譜記錄
Resources 有用物資 白紙、空表、譜紙
Unnamed 無名曲譜 -

操作員可按需要加入或移除類別。更多資訊於後面「操作」部份再有說明。

# 相關連結

  • 取出類別 或 放入類別
  • 在一個類別文件夾中取/放內容，或檢查文件夾：快速行動
  • 加入類別，或，在較罕見的情況，移除類別
  • 加入狀態 或 撤回狀態
